# HSJ-COLLECTION
> Collection related classes for the Hubrisjs javascript framework.

Some command classes: ArrayMap, SparseArray, Bst, RingBuffer, Queue etc...

## Installation

Node:

```sh
npm install hjs-collection --save
```
